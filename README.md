# Parallel Text Processing with Multiprocessing

## Overview
This project compares Sequential Processing and Multiprocessing 
for customer review sentiment analysis.

## Features
- Sentiment Score System (+1 / -1)
- Rule-based classification
- Sequential vs Multiprocessing comparison
- Execution time measurement
- Parallel execution visualization using Process IDs

## Technologies Used
- Python
- Multiprocessing
- Time module

## How to Run
1. Add your text files
2. Update file paths in the script
3. Run:

   python parallel_processing.py
